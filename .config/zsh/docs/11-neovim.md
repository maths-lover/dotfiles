# 11. Neovim

The Neovim config has its own full documentation set, alongside the config at
`~/.config/nvim`:

- [Neovim docs index](../../nvim/docs/README.md)
- [Neovim README](../../nvim/README.md)

It covers getting started, architecture, keymaps, plugins, LSP & languages,
completion, navigation, git, theme sync, Neovide, and troubleshooting.

In short: a modular lazy.nvim config that follows the OS `theme` switcher, with LSP
for many languages and Neovide GUI support.
